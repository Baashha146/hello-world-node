# hello-world-node

Simple "Hello World" sample project using HTML, CSS and a Node (Express) server.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open `http://localhost:3000` in your browser.

## Files
- `server.js` — simple Express static file server
- `public/index.html` — homepage
- `public/about.html` — example linked page (shows `href` usage)
- `public/styles.css` — stylesheet
- `package.json` — project metadata & script
